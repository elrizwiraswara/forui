<Callout type="info" title="Not a searchable select">
    An autocomplete is not a searchable select. it is a text-field with suggestions. Values are not limited to one of
    suggestions, users can type anything. If you need a searchable select, use
    [a searchable select](/docs/widgets/form/select#searchable)
    or [a searchable multi select](/docs/widgets/form/multi-select#searchable)
</Callout>

## CLI

To generate a specific style for customization:

<Tabs items={['Autocomplete', 'Autocomplete Field Sizes', 'Autocomplete Field', 'Autocomplete Content', 'Autocomplete Section']}>
    <Tab value="Autocomplete">
        ```shell copy
        dart run forui style create autocomplete
        ````
    </Tab>
    <Tab value="Autocomplete Field Sizes">
        ```shell copy
        dart run forui style create autocomplete-field-sizes
        ````
    </Tab>
    <Tab value="Autocomplete Field">
        ```shell copy
        dart run forui style create autocomplete-field
        ````
    </Tab>
    <Tab value="Autocomplete Content">
        ```shell copy
        dart run forui style create autocomplete-content
        ````
    </Tab>
    <Tab value="Autocomplete Section">
        ```shell copy
        dart run forui style create autocomplete-section
        ````
    </Tab>
</Tabs>
