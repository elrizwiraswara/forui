## CLI

To generate a specific style for customization:

<Tabs items={['Multi Select', 'Multi Select Field Sizes', 'Multi Select Field', 'Multi Select Tag']}>
    <Tab value="Multi Select">
        ```shell copy
        dart run forui style create multi-select
        ````
    </Tab>
    <Tab value="Multi Select Field Sizes">
        ```shell copy
        dart run forui style create multi-select-field-sizes
        ````
    </Tab>
    <Tab value="Multi Select Field">
        ```shell copy
        dart run forui style create multi-select-field
        ````
    </Tab>
    <Tab value="Multi Select Tag">
        ```shell copy
        dart run forui style create multi-select-tag
        ````
    </Tab>
</Tabs>
