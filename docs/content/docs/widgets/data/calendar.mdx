The calendar supports swipe gestures on mobile platforms, allowing users to navigate between pages by swiping left or right.

A [`FCalendarControl`](https://pub.dev/documentation/forui/latest/forui.widgets.calendar/FCalendarControl-class.html)
controls navigation, while a [`FDateSelectionControl`](https://pub.dev/documentation/forui/latest/forui.widgets.calendar/FDateSelectionControl-class.html)
controls the selection behavior (single date, multiple dates, or a range).

<Callout type="info">
    `FCalendar`, `FDateSelectionController`, and all `FCalendarController`s return `DateTime`s in UTC timezone, truncated
    to the nearest day.
</Callout>
